2.1. DOM

2.1.2. Tương tác với thẻ HTML
- Tương tác thông qua id
    + document.getElementById('tên_id')
- Tương tác thông qua class
    + document.getElementByClass('tên_class')
- Tương tác thông qua tên thẻ HTML
    + document.getElementByTagName('tên_thẻ_HTML')
- Tương tác thông qua cú pháp CSS Selector
    + document.querySelectorAll('cú_pháp_CSS_selector')
